# outlook-mail-forward-to-backlog

Outlookに届いたメールを片っ端からBacklogに転送するやつ。  
カスタマイズすれば特定のメールのみを転送するようにすることも可能。

メールの中身を極力いろんなサービスを経由させないように、Azure Logic Apps上でアプリを構築した。

## 導入方法

そのうち書く。  
Parametersは自分で設定すること。

ちなみに、会話IDを置く場所がなかったので件名に置いている。  
カスタム属性に入れるように修正が必要。
